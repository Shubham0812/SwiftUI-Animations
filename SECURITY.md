# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Please do NOT open a public GitHub issue for security vulnerabilities.**

Instead, send an email to **singh.shubham0812@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce the issue
- Any potential impact

You can expect an initial response within 48 hours. We take all security reports seriously and will work with you to understand and address the issue.

## Scope

This project is a collection of SwiftUI animation examples intended for educational and reference purposes. While it does not handle sensitive data or network requests, we still want to ensure the codebase is safe and free from any malicious code.
