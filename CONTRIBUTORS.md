## Want to See Your Name Here?

We welcome contributions of all kinds — new animations, bug fixes, documentation improvements, and more.

1. Read the **[Contributing Guide](CONTRIBUTING.md)** to get started
2. Submit a pull request with your changes
3. Once merged, you'll be added to this list!

Every contribution matters, no matter how small. Thank you for helping make SwiftUI Animations better for everyone.
